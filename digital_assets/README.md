# Digital Assets Demo

An interactive Streamlit application for digital-asset / crypto calculations:

- **Portfolio Tracker** — enter holdings and track current value with live or manual prices
- **DeFi Yield Calculator** — compare APR vs APY and project earnings over time
- **Tokenomics Modeller** — model token supply, emission schedule, and market-cap projections

## Setup (UV)

Use these commands every time:

```bash
brew install uv
cd digital_assets
uv sync
```

## Run

```bash
uv run streamlit run app.py
```
