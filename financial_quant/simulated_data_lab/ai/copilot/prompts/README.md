# Copilot prompt workspace

Store reusable prompt templates for GitHub Copilot here.

Suggested files:
- `dataset_planning.prompt.md`
- `feature_engineering.prompt.md`
- `validation_checks.prompt.md`

Template starter:

```markdown
# Goal
Describe the exact dataset or transformation to build.

# Constraints
List schema, row count target, memory limits, and output format.

# Acceptance criteria
Define checks that must pass before completion.
```
