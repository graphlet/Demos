# Copilot prompt workspace

Store reusable prompt templates for GitHub Copilot here.

Suggested files:
- `var_simulations_planning.prompt.md`
- `backtest_validation.prompt.md`
- `scenario_stress.prompt.md`

Template starter:

```markdown
# Goal
Describe the exact simulation or analysis workflow to build.

# Constraints
List schema, data horizon, confidence levels, and output expectations.

# Acceptance criteria
Define checks that must pass before completion.
```
